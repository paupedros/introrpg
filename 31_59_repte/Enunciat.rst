
``Enunciat``

Volem dibuixar una forma geomètrica amb angles rectes (rectangle/quadrat), però volem que sigui versàtil, l'usuari hauria de poder decidir la seva amplada i la llargada:


.. code-block:: java
	
	java Repte
	Quina amplada vols?
	10
	Quina llargada vols?
	5
